---
description: Look up Design Authority decisions
argument-hint: <DA-ID or search-term or "pending">
---

If input is a DA ID (e.g., "DA-0001"):
- Return the full decision from docs/decisions/DA-XXXX.md

If input is "pending":
- List all files in docs/decisions/pending/

If input is a search term:
- Search all decision files for the term
- Return matching decisions with their IDs
